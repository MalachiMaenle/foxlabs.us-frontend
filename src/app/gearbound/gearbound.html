Installation Guide

Curseforge installation guide

  1. Download the zip file above
  2. Open the curseforge app
  3. Click Import on the top left of the screen
  4. Then click "Choose .Zip file"
  5. Select the zip file you just downloaded
  6. And thats it well done

Modrinth installation guide

  1. Download the mrPack above
  2. Open the modrinth app
  3. Click Create new instance on the middle left of the screen
  4. now choose From File
  5. and click Import from file

Prism installation guide

  1. Download the zip file above
  2. Open Prism Launcher
  3. Click "Add Instance" (top left)
  4. Select "Import from zip" and browse for the file you downloaded
  5. Click "OK" and let it download the mods
  6. Double click the new instance to play
